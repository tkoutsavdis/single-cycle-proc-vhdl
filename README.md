# Single-Cycle Processor (VHDL) — CHARIS ISA

VHDL implementation of a **single-cycle (non-pipelined) processor** designed for the CHARIS instruction set.

## Documentation
- Assignment: `Assignment-1.pdf`
- Report: `REPORT.pdf`

## Repository contents
- `SOURCES/` — VHDL modules (ALU, Register File, datapath stages, control, memory, etc.)
- `WAVEFORMS/` — simulation waveform configuration files (`.wcfg`) used in Xilinx ISim/Vivado simulation.
